"Crea una aplicación web SPA (Single Page Application) utilizando React y Bootstrap 5. La aplicación es para un taller mecánico en Santiago de Chile especializado en frenos. El diseño debe ser profesional, limpio, moderno y responsivo, utilizando una paleta de colores industrial (gris oscuro, blanco y detalles en azul o rojo automotriz).

Requerimientos Técnicos:

Persistencia Local: La app debe funcionar de forma asíncrona y guardar todo en localStorage. Si la página se recarga o se cierra, los datos no deben perderse (simulando una base de datos local tipo 'Room').

Navegación: Un menú lateral o superior con 3 vistas:

Gestión de Inventario: Listado y búsqueda de productos.

Registro de Compras: Formulario para ingreso masivo de stock.

Órdenes de Trabajo: Creación y visualización de servicios.

Detalle de Vistas:

Gestión de Inventario (Frenos):

Debe permitir ver y filtrar productos.

Campos: N° de freno, Marca de auto compatible, Modelo específico, Año, Proveedor, Cantidad (stock) y Calidad (Select: Excelente, Buena, Regular, Mala).

Incluir una barra de búsqueda rápida por patente o modelo.

Registro de Compras:

Optimizado para entradas grandes (el taller compra de 4,000 a 6,000 pastillas).

Simular un botón de 'Importar Excel' y un formulario rápido de carga manual.

Órdenes de Trabajo (OT):

Formulario con los campos: Nro OT (correlativo), Patente (formato chileno), Marca, Modelo, KM, Fecha, Total, Realizado por, Revisado por, y un campo de Firma del cliente.

Checkboxes para servicios (pueden ser nulos): Cambio pastillas, cambio balatas, cambio líquido, cambio gomas, rectificado, sangrado, cambio piola, revisión, otros.

Mostrar un listado de las OTs creadas con opción de ver detalles.

Estilo Visual:

Usa componentes de Bootstrap para tablas y formularios.

Agrega iconos de Lucide-React o FontAwesome.

El fondo de la app debe ser light gray (#f4f4f4) para que las tarjetas blancas (cards) de contenido resalten.

Asegúrate de que las tablas sean hoverable y los botones tengan bordes redondeados (rounded-pill).

Genera el código completo en una estructura de carpetas organizada (components, hooks, views)."
